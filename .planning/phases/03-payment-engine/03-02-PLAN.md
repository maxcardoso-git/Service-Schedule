<task type="auto">
  <name>Task 1: Create payments route file with three endpoints</name>
  <files>src/routes/payments.js</files>
  <action>
Create src/routes/payments.js following the exact same pattern as src/routes/bookings.js.

Imports:
- Router from 'express'
- z from 'zod'
- apiKeyAuth from '../middleware/auth.js'
- validate from '../middleware/validate.js'
- createPixIntent, getPaymentStatus, simulatePaid from '../services/paymentService.js'

Setup:
- `const router = Router();`
- `router.use(apiKeyAuth);` (all endpoints behind API key auth)
- Define asyncHandler wrapper (same pattern as bookings.js)

Three endpoints:

1. **POST /pix** (PYMT-01 — create PIX payment intent):
   - validate body: `{ bookingId: z.string().uuid() }`
   - Call createPixIntent(req.body.bookingId)
   - Return 201 with `{ data: payment }`

2. **GET /:id/status** (PYMT-02 — get payment status):
   - validate params: `{ id: z.string().uuid() }`
   - Call getPaymentStatus(req.params.id)
   - Return 200 with `{ data: payment }`

3. **POST /:id/simulate-paid** (PYMT-03 — simulate payment confirmation):
   - validate params: `{ id: z.string().uuid() }`
   - Call simulatePaid(req.params.id)
   - Return 200 with `{ data: payment }`

Export default router.
  </action>
  <verify>
File exists at src/routes/payments.js.
`node -e "import('./src/routes/payments.js').then(() => console.log('OK'))"` prints OK without errors.
  </verify>
  <done>payments.js exports a Router with 3 endpoints, all behind apiKeyAuth, with Zod validation on inputs.</done>
</task>
