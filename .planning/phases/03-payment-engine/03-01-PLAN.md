<task type="auto">
  <name>Task 2: Create payment service layer with three business functions</name>
  <files>src/services/paymentService.js</files>
  <action>
Create src/services/paymentService.js with three exported functions:

1. **createPixIntent(bookingId)**:
   - findUnique booking by id, include: { services: true, payment: true }
   - If not found: throw NotFoundError('Booking not found', 'BOOKING_NOT_FOUND')
   - If booking.status !== 'CONFIRMED': throw ValidationError('Payment intent can only be created for CONFIRMED bookings', null, 'BOOKING_NOT_CONFIRMED')
   - If booking.payment exists: throw ConflictError('Payment already exists for this booking', 'PAYMENT_ALREADY_EXISTS')
   - Sum amount from booking.services using: `booking.services.reduce((sum, bs) => sum + parseFloat(bs.price), 0)` then `Number(total).toFixed(2)` for clean decimal string
   - Generate PIX payload: `PIX-SIM:txid=${uuid-no-dashes-uppercase}:booking=${bookingId}:amount=${amount}`
   - prisma.payment.create with { bookingId, amount, status: 'PENDING', pixPayload }
   - Wrap create in try/catch: catch P2002 (err.code === 'P2002') and re-throw as ConflictError('Payment already exists for this booking', 'PAYMENT_ALREADY_EXISTS')

2. **getPaymentStatus(paymentId)**:
   - findUnique by id
   - If not found: throw NotFoundError('Payment not found', 'PAYMENT_NOT_FOUND')
   - Return full payment object

3. **simulatePaid(paymentId)**:
   - findUnique by id
   - If not found: throw NotFoundError('Payment not found', 'PAYMENT_NOT_FOUND')
   - If payment.status !== 'PENDING': throw ConflictError(`Cannot simulate payment in status ${payment.status}`, 'INVALID_STATUS_TRANSITION')
   - prisma.payment.update where id, data: { status: 'PAID' }
   - Return updated payment

Imports:
- `import { v4 as uuidv4 } from 'uuid';`
- `import prisma from '../lib/prisma.js';`
- `import { NotFoundError, ValidationError, ConflictError } from '../lib/errors.js';`

Helper function generatePixPayload(bookingId, amount) at module scope (not exported).
