<task type="auto">
  <name>Task 1: Express static serving + SPA fallback</name>
  <files>src/app.js</files>
  <action>
1. In src/app.js, add static file serving AFTER all API route mounts, BEFORE the global error handler:

   ```javascript
   import path from 'path';
   import { fileURLToPath } from 'url';

   const __dirname = path.dirname(fileURLToPath(import.meta.url));

   // Serve frontend build in production
   if (process.env.NODE_ENV === 'production') {
     const distPath = path.join(__dirname, '..', 'frontend', 'dist');
     app.use(express.static(distPath));

     // SPA fallback: non-API routes serve index.html
     app.get('*', (req, res, next) => {
       if (req.path.startsWith('/api')) return next();
       res.sendFile(path.join(distPath, 'index.html'));
     });
   }
   ```

2. VERIFY: The global error handler (`app.use(errorHandler)`) MUST remain as the very last middleware. The static serving goes between API routes and error handler.

3. VERIFY: The SPA fallback checks `req.path.startsWith('/api')` to prevent API 404s from returning HTML.

4. If `path` and `fileURLToPath` are not already imported at the top of app.js, add the imports. Check existing imports first to avoid duplicates.

5. Build the frontend to test:
   ```bash
   cd frontend && npm run build
   ```
   Verify frontend/dist/index.html exists.
